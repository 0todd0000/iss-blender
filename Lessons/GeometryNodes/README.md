# Overview

Geometry nodes (GN) are graphical programming units that create and/or modify geometry.

This tutorial explains the basics of geometry nodes.  It is a simplified version of the  [Geometry Nodes for Complete Beginners](https://www.youtube.com/watch?v=tWvgHbZXCtA&list=PLsGl9GczcgBsDZ3jQPwP1p6evNI22wevP) tutorial.
